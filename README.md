# Cafe-AGENTS-UI

An Android application for managing cafe agents using a chatbot interface.

## Project Structure

The project follows a clean architecture pattern with Jetpack Compose for the UI.

- `ui/`: Contains all the Compose screens and components.
- `viewmodel/`: Contains ViewModels for state management.
- `model/`: Data models used throughout the app.
- `repository/`: Data layer for handling business logic and data fetching.
- `network/`: Retrofit setup for API communication.
- `navigation/`: Navigation graph for screen transitions.
- `utils/`: Utility classes and constants.
- `theme/`: App-wide styling and themes.

## Getting Started

1. Sync the project with Gradle.
2. Ensure you have the latest Android Studio installed.
3. Run the app on an emulator or a physical device.
